Redes y Protocolos

Las redes y los protocolos permiten la comunicación eficiente y segura en la web, una parte fundamental del desarrollo web.

¿Qué son las Redes y Protocolos?


Las redes permiten la conexión entre dispositivos para el intercambio de datos, mientras que los protocolos definen las reglas para esa comunicación. Son la base del funcionamiento de Internet y de todas las aplicaciones web.

Protocolos Fundamentales

TCP/IP

Protocolo base de la comunicación en Internet. Divide los datos en paquetes que son enviados y reensamblados en el destino.

HTTP/HTTPS

Protocolo utilizado para la transferencia de datos entre navegadores y servidores web. HTTPS asegura los datos con cifrado.

FTP

Protocolo para transferir archivos entre servidores y clientes. Se utiliza en procesos de despliegue y transferencia de contenido.

DNS

Convierte nombres de dominio legibles por humanos (como www.ejemplo.com) en direcciones IP comprensibles para las máquinas.

Conceptos Clave

Firewall

Dispositivo o software que protege redes bloqueando accesos no autorizados mientras permite el tráfico legítimo.

VPN

Red privada virtual que encripta la conexión y protege la privacidad del usuario en redes públicas.

WebSocket

Protocolo para comunicación bidireccional en tiempo real, ideal para aplicaciones como chats y videojuegos.

ICMP

Protocolo utilizado para diagnosticar problemas de red, como el comando "ping".

Ejemplo: Petición HTTP Básica

Este es un ejemplo donde se muestra cómo realizar una petición HTTP a un servidor desde Node.js.

const https = require('https');
https.get('https://jsonplaceholder.typicode.com/posts/1', (resp) => {
    let data = '';

    // Recibir datos
    resp.on('data', (chunk) => {
        data += chunk;
    });

    // Finalizar respuesta
    resp.on('end', () => {
        console.log(JSON.parse(data));
    });

}).on("error", (err) => {
    console.log("Error: " + err.message);
});
